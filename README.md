
# fleek

 it predicts loan payback ability accurate to 95%
 it predicts behavaroural partern accurate to 80-85% 

This whole repo is webhooked on heroku at [fleek-api.](https://fleek-api.herokuapp.com/) 
Test route is /predict {post}
more info is in the raw Ai files(contains raw version, jupiternotebook, dataset, etc)

The UI/Web can be viewed at https://samuel-ibits.github.io/fleek/web/code/html/index.html
